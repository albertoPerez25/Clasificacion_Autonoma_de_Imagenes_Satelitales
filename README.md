# Clasificacion_Autonoma_de_Imagenes_Satelitales

Predicción de una clasificación de imágenes satélitales geográficas de baja resolución del dataset GeoSat. Uso de Keras, TensorFlow, modelos de Hugging Face e implementado en Python.

- Creación de una red CNN desde cero.
- Fine-Tuning mediante Transfer-Learning a modelos de estado del arte.
- Uso de MLPs y Vision Transformers (ViT).

En otras ramas se implementó:
- Modelos de Machine Learning tradicional.
  - Regresión Logística, Naive Bayes, Árboles de decisión...
- Ensembles
  - Gradient Boosting, AdaBoost, Random Forest...  
- Realización de un EDA completo.
- Feature Engineering.
